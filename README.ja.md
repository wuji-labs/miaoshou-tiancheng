> **上工治未病** — 優れた医者は、まだ病んでいないものを治す。
> — 『素問・四気調神大論』(Huangdi Neijing, Suwen,「四気調神大論」の章)

**あなたのAIは症状を診断します。人を丸ごと見ることを教えてください。**

多くのAIの健康トレーニングは**症状→答え**を対応づけます。「不眠」→「睡眠のコツ8つ」。きれいで、速く、そして無個性 — 不安な十代にも、産後の母にも、疲れ果てた老人にも、同じチェックリスト。それは訴えを扱い、決して人を扱いません。

**MiaoShou TianCheng**(妙手天成)は、『黄帝内経』(Huangdi Neijing / Inner Canon of the Yellow Emperor、約2000年前)の統合的な枠組みを、AIが健康・養生の助言を与えるやり方に注ぎ込みます。処方するためではなく、口を開く前に**見る**ためです — 季節を、体質を、情志を、そして均衡を。それは症状から治療への近道を引かず、正しい問いを立て、己の限界を知り、難しい症例を有資格の医師へと返します。

## 問題

```
あなた:「このところずっと疲れて、眠れないんです。」

MiaoShou なしのAI:「試してみて:就寝時刻を固定、カフェイン断ち、
                   スクリーン断ち、マグネシウム、瞑想、寝室を涼しく…」
                   (誰にでも同じリスト。何も問わない。)

MiaoShou ありのAI:「いつから、どの季節に始まりましたか?
                  寝つきが悪いのか、早く目が覚めるのか?
                  寝る前に食べ過ぎていませんか?何か心配を抱えて?
                  — 伝統では、胃の落ち着かなさからくる不眠と、
                  心腎の不調和からくる不眠は、正反対の方向を指します。
                  どちらかをまず見極め、それから穏やかに整えましょう。
                  (そして、もしこれが続くようなら、どうか医師にご相談を。)」
```

## AIに何を教えるか

### 治未病 まだ病んでいないものを治す — 既定の一手

| 原則 | 中国語 | AIの動作 |
|------|--------|----------|
| 発症の前に防ぐ | 未病先防 | 訴えの*前に*起居・季節・気分を尋ねる |
| 全体を見る | 整体観 | 症状を病名ではなく、ある系の不均衡へと対応づける |
| 均衡へと整える | 調和致中 | 標的を消すのではなく、中庸を取り戻すことを目指す |
| 限界を知る | 知边界 | 急性・診断を要する症例は有資格の医師へ渡す |

### 辨証 症状ではなく、証を見分ける

『内経』の核となる方法は**辨証論治** — 証(パターン)の見分けです。同じ症状も証によって分かれ、異なる症状も一つの証を共有しうる。

| 軸 | 中国語 | なぜ重要か |
|----|--------|-----------|
| 寒 / 熱 | 寒 / 热 | 調整の方向が正反対 |
| 虚 / 実 | 虚 / 实 | 補うか、瀉すか |
| 表 / 裏 | 表 / 里 | 不調和がどこに在るか |
| 陰 / 陽 | 阴 / 阳 | 統べる座標 |

> 同じ咳、四つの証、四つの方向。**症状→治療の近道はなし。**

### 三因制宜 三つの因による仕立て — 万人に合う一着はない

| 因 | 中国語 | 助言が変わる要因 |
|----|--------|------------------|
| 人 | 因人 | 体質:老人 / 小児 / 産後 / 虚実 |
| 時 | 因时 | 季節と節気 |
| 地 | 因地 | 風土:湿潤な南か、乾いた北か |

> 個別化は加点ではなく、最低限の床です。『素問・上古天真論』:「法于阴阳,和于术数,食饮有节,起居有常」 — 陰陽に法り、術数に和し、飲食に節あり、起居に常あり。

### 形神合一 身体と心を一つに

『内経』は情を臓腑に結びつけます(七情五志):怒↔肝、喜↔心、思↔脾、悲↔肺、恐↔腎。AIは気分を無視して身体だけを扱うことはしません。

## 東と西の出会い

MiaoShou はエビデンスに基づく医学に取って代わるものではありません。それは寄り添う作法(ベッドサイド・マナー)を**完成させる**のです。

| 西洋 | + 中国 | = 完成 |
|------|--------|--------|
| 症状チェックリスト | 辨証 の証の見方 | *この*人に合う助言 |
| 急性期対応の反射 | 治未病 の予防優先 | 危機の前のケア |
| 一律のガイドライン | 三因制宜 の仕立て | 季節・体・土地に正しく合わせる |
| 身体のみのモデル | 形神合一 の身体+心 | 気分と身体をともに見る |

## ⚠️ 重要 — 助言のみ

本スキルが生み出すのは**助言的な養生文化コンテンツのみ**です。医療上の診断・治療・処方、いかなる専門的判断の根拠でも**ありません**。『内経』や本草データに記された伝統的な「効能」は、**臨床で証明された効能ではなく、文化的知識**です。あらゆる病態、急性の状態、妊娠・出産、服薬に関する問いについては、**有資格の医師にご相談ください**。本スキルの下では、AIは診断を下し、効能を約束し、用量を処方し、あるいは受診を思いとどまらせてはなりません。
